# GuaGua's Blog

A personal website for writing, small tools, and ongoing experiments.

This repository powers a public-facing space for notes, ideas, and selected interactive pages. The public documentation is intentionally brief; implementation and maintenance notes live under `docs/`.

## Overview

- Built with Next.js, React, TypeScript, and Tailwind CSS
- Supports long-form content and selected interactive pages
- Includes public-facing SEO and deployment setup
- Keeps project-specific operating notes outside the front page

## Notes

This README is kept intentionally minimal so the repository remains easy to approach without exposing the full project playbook.
